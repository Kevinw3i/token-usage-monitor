用量小工具 · 九種樣式

每個小工具一次顯示「一個模型」,可點上方/下方分頁切換不同 provider。每頁都看得到 5H 用量、1W 用量、當前模型與方案。資料為示意值。

↳ 試著點每個小工具的分頁切換 Codex / Claude / Gemini / Grok。Grok 故意設為 API 計費(無方案),示範「沒有方案」時的呈現。
~/usage --watch
STYLE 01
Terminal CRT
磷光綠/琥珀、ASCII 進度條、掃描線、頂部 ▸ 分頁。
適合:常駐終端機旁、hacker 美學。
STYLE 02
Frosted Glass
毛玻璃透出桌布、5H/1W 雙環、底部圓點分頁。
適合:放桌面角落、與桌布融合。
STYLE 03
Swiss Brutalist
粗黑框、超大 5H 數字、分頁是反黑標籤、紅色警示段。
適合:愛極簡衝擊、一眼讀數字。

// SYSTEM LOAD

STYLE 04
Neon Synthwave
霓虹發光長條、賽博網格、發光分頁名。
適合:桌面要有個性、喜歡炫光。
STYLE 05
Luxury Instrument
炭灰金屬、5H/1W 金/銅弧形儀表、Fraunces 襯線、底部金點分頁。
適合:偏好精緻儀表板質感。
STYLE 06
Pixel Arcade
8-bit 像素風、HP/MP 血條隱喻用量、Press Start 字體。
適合:遊戲魂、想要玩心十足的桌面。
STYLE 07
Quant Terminal
彭博式黑底密集數據、用量依高低變綠/黃/紅、橙色點綴。
適合:盯盤/交易終端美學、要警示色。
STYLE 08
Paper / E-ink
米白紙感、Newsreader 襯線、細墨線進度、安靜優雅(淺色)。
適合:喜歡安靜、紙本質感、淺色桌面。
STYLE 09
Bento SaaS
現代產品風、圓角便當格、頂部 segment 切換、清爽柔影(淺色)。
適合:偏好乾淨現代產品 UI、日常實用。
